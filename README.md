# Credit Card Validator

## Описание

CLI-приложение на Go для проверки номеров банковских карт.
Проект выполняет валидацию номера карты по алгоритму Луна и определяет банк по префиксу карты из файла `banks.txt`.

## Возможности

* Проверка номера карты по алгоритму Луна
* Определение банка по префиксу карты
* Проверка корректности пользовательского ввода
* Поддержка номеров длиной от 13 до 19 цифр

## Установка и запуск

### Клонировать репозиторий

```bash
git clone <repository_url>
cd credit-card-validator
```

### Запустить приложение

```bash
go run main.go
```

## Примеры использования

```text
Введите номер карты (или Enter для выхода): 4000000000000002

Банк: Lunar Bank
```

```text
Введите номер карты (или Enter для выхода): 4000000000000001

Ошибка: номер не прошёл проверку Луна
```
